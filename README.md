# 사용자 음식 사진 기반 맛집 추천 앱

## 프로젝트 명
오늘 뭐먹지?

## 프로젝트 배경
**기획 배경**
낯선 곳에 가서 밥을 먹어야 할 때, 보통은 딱히 어떤 메뉴가 끌리지 않을 때가 많다. 그때 우리는 "oo 맛집"이라고 포털에 검색해 맛집 리스트들을 끝없이 탐색한다.
그러면서도 식당에 가서 음식이 나오면 항상 사진을 찍어 보관할 정도로 음식에 대한 관심이 많다.
따라서 사용자들이 찍은 음식 사진들을 분석해 분류하고, 그들이 찍은 사진을 바탕으로 사용자가 선호하는 음식이 있는 주변 식당 추천 앱을 기획하게 되었다. 

**기능**
* 사용자 사진 분석 밎 분류
* 근처 식당 파악 밎 추천
* 식사 시간 때마다 push알림으로 메뉴 추천

**유사 앱과의 차별점 및 기회요소**
기존의 맛집 앱은 단순히 주변 맛집 리스트 만을 보여준다.

**구성원 별 역할**

|학번 | 이름 | 맡은일
---------------------
|20171614 |김세연 | 기획|

**프로젝트 구현**
* 음식 분류 관련 오픈소스를 찾아 적절히 활용한다
* 식당 추천은 지도 api 활용할 예정
